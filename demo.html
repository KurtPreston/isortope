isortope

Simple, animated tablesorting with support for numbers, fields, currency, and percents

How to use

Load jQuery and Isotope before loading isortope: 
<script src='jquery.js' />
<script src='isotope.js' />
<script src='isortope.js' />
Then, initialize isortope one of two ways:
1. Either add the isortope class to any table: 
<table class='isortope' >
  ...
</table>
2. Or, call the .isortope() method on any jQuery object. This can be useful for tables loaded over AJAX. 
$('table#my-table').isortope();
If you want animation, add the Isope animation styles to your stylesheet: 
.isotope,
.isotope .isotope-item {
  transition-duration: 0.8s;
}

.isotope {
  transition-property: height, width;
}

.isotope .isotope-item {
  transition-property: transform, opacity;
}

.isotope.no-transition,
.isotope.no-transition .isotope-item,
.isotope .isotope-item.no-transition {
  transition-duration: 0s;
}
